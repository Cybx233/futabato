# 双叶妹妹

Chrome/Edge 浏览器扩展，把公众号文章中的图片还原成接近纸质书的左右双页阅读模式。

纯离线运行，无网络请求，不收集任何数据。

![阅读模式截图](Light-browser.png)

## 功能特性

- **双页 / 单页自适应** — 宽屏自动双页跨页，窄屏（< 900px）自动切单页，大屏小屏都好读
- **阅读方向切换** — 默认 RTL（日漫从右往左），一键切 LTR（欧美从左往右）
- **奇偶跨页偏移** — 跨页对不齐时，插入空白首页重新对齐左右页
- **快捷键启动** — `Cmd+Shift+Y`（Mac）/ `Ctrl+Shift+Y`（Win）直接切换阅读模式
- **多种翻页模式** — 点击屏幕左右两侧翻页 /方向键 ← → 翻页，Esc 退出/鼠标滚轮上下滚动翻页
- **图片过滤** — 两阶段过滤头像、表情、横幅等非漫画图片（基于 `data-w` 与自然尺寸）
- **合集导航** — 自动检测文章合集，在底部显示上一篇/下一篇
- 加载失败可点击重试，不影响其他页面
- 纯离线，所有逻辑本地运行，无任何网络请求

## 安装

**推荐方式：前往 [Releases](https://github.com/Cybx233/Futabato/releases) 下载最新版 `双叶妹妹.zip`，解压后按以下步骤加载。**

也可以直接克隆仓库：

```bash
git clone https://github.com/Cybx233/Futabato.git
```

下载或克隆后：

1. 打开 Chrome，地址栏输入 `chrome://extensions`。
2. 右上角开启「开发者模式」。
3. 点击「加载已解压的扩展程序」，选择本仓库根目录（含 `manifest.json` 的那个文件夹）。
4. 扩展出现在列表中即安装成功。

> 也支持 Edge（`edge://extensions`）等 Chromium 内核浏览器，步骤相同。

## 使用

1. 用浏览器打开任意一篇包含漫画的公众号文章。
2. 点击工具栏上的扩展图标，或按 `Cmd+Shift+Y`（Mac）/ `Ctrl+Shift+Y`（Win）快捷键。
3. 进入全屏阅读模式，可点击屏幕两侧、按方向键或滚动鼠标滚轮翻页。按 Esc 退出。

如果点击图标后扩展没反应（橙色 `!` 角标），说明文章页是在安装扩展前打开的，刷新一下页面即可。

## 权限说明

完全本地运行 — 不发起任何网络请求，不收集任何数据。

## 技术实现

- Manifest V3 架构，`background.js` 为 service worker，负责监听工具栏点击并转发消息。
- `content.js` 为核心，以 IIFE + 单例 `ReaderController` 组织，用 **Shadow DOM** 把阅读器 UI 与原页面样式完全隔离，避免互相污染。
- 所有样式内联在 `_getCSS()` 中，无外部 CSS 文件、无构建步骤、无第三方依赖。
- 图片识别基于图片自然尺寸的比例判断（全景图 / 长条图 / 普通页）。
