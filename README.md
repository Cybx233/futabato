# 双叶妹妹

 Chrome/Edge 浏览器扩展，把公众号文章中的图片，还原成接近纸质书的左右双页阅读模式。
 
 纯离线运行，无网络请求，不收集任何数据。

![阅读模式截图](screenshot.jpg)

## 功能特性

- **双页 / 单页自适应**：宽屏自动双页跨页展示，窄屏（< 900px）自动切单页。
- **阅读方向切换**：默认 RTL（日漫从右往左），可一键切换为 LTR（欧美从左往右）。
- **奇偶跨页偏移**：跨页对不齐时，用「奇偶切换」在首页插入空白页，重新对齐左右页。
- **多种翻页方式**：点击屏幕左右两侧翻页、方向键翻页、鼠标滚轮翻页、Esc 退出。
- **合集上下篇导航**：自动检测文章合集，在进度条下方显示上一篇/下一篇，适配阅读方向。
- **进度条与页码**：绿色进度条，简洁的组号/总组数页码，点击画面切换显示。
- **全景图与长条图识别**：横向全景图自动放宽显示，竖向长条图给出提示。
- **图片智能过滤**：两阶段过滤掉头像、表情、横幅等非漫画图片（按 `data-w` 与自然尺寸）。
- **加载失败重试**：单独某页加载失败时可点击重试，不影响整体阅读。
- **快捷键启动**：`Cmd+Shift+Y`（Mac）/ `Ctrl+Shift+Y`（Win）直接切换阅读模式。
- **纯离线**：所有逻辑本地运行，不向任何服务器发送数据。

## 安装

1. 下载或克隆本仓库到本地。
2. 打开 Chrome，地址栏输入 `chrome://extensions`。
3. 右上角开启「开发者模式」。
4. 点击「加载已解压的扩展程序」，选择本仓库根目录（含 `manifest.json` 的那个文件夹）。
5. 扩展出现在列表中即安装成功。

> 也支持 Edge（`edge://extensions`）等 Chromium 内核浏览器，步骤相同。

## 使用

1. 用浏览器打开任意一篇包含漫画的微信公众号文章（`mp.weixin.qq.com/s...`）。
2. 点击工具栏上的扩展图标，或按 `Cmd+Shift+Y`（Mac）/ `Ctrl+Shift+Y`（Win）快捷键。
3. 进入全屏阅读模式，可点击屏幕两侧、按方向键或滚动鼠标滚轮翻页。按 Esc 退出。

如果点击图标后扩展没反应（橙色 `!` 角标），说明文章页是在安装扩展前打开的，刷新一下页面即可。

## 权限说明

| 权限 | 用途 |
|------|------|
| `mp.weixin.qq.com` 主机权限 | 仅在公众号文章页注入阅读器界面、读取文章中的图片。不访问其他网站，不向任何服务器发送数据。 |

扩展没有申请任何额外的隐私权限（如标签读取、浏览历史、Cookie 等），所有处理都在本地浏览器内完成。

## 技术实现

- Manifest V3 架构，`background.js` 为 service worker，负责监听工具栏点击并转发消息。
- `content.js` 为核心，以 IIFE + 单例 `ReaderController` 组织，用 **Shadow DOM** 把阅读器 UI 与原页面样式完全隔离，避免互相污染。
- 所有样式内联在 `_getCSS()` 中，无外部 CSS 文件、无构建步骤、无第三方依赖。
- 图片识别基于图片自然尺寸的比例判断（全景图 / 长条图 / 普通页）。
